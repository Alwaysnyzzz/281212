NyzzUploader API

API upload file publik yang simpel. Upload file apapun, dapatkan URL pendek permanen — cocok untuk bot WhatsApp, Telegram, atau integrasi apapun.

⚡ REST API ∞ No Expired ⚠ Maks 3 MB 📦 All Format
01

Base URL

url 
https://uploader.nyzz.my.id
ℹ️ Semua request tidak memerlukan API key. API ini bersifat publik dan gratis.

Integrasi Bot WhatsApp

📦 Contoh menggunakan Baileys (library WA paling populer). Install: npm install @whiskeysockets/baileys
1
Terima media dari pengguna
Saat bot menerima pesan berisi gambar/video/dokumen, download buffer-nya lalu upload ke NyzzUploader.
javascript — Baileys
import { downloadMediaMessage } from '@whiskeysockets/baileys'
import FormData from 'form-data'
import fetch from 'node-fetch'

const UPLOAD_URL = 'https://uploader.nyzz.my.id/api/upload'

// Fungsi upload buffer ke NyzzUploader
async function uploadToNyzz(buffer, filename) {
  const form = new FormData()
  form.append('file', buffer, { filename })

  const res = await fetch(UPLOAD_URL, {
    method: 'POST',
    body: form,
    headers: form.getHeaders()
  })

  const json = await res.json()
  if (!json.success) throw new Error(json.error)
  return json.url
}

// Handler pesan di Baileys
sock.ev.on('messages.upsert', async ({ messages }) => {
  const msg = messages[0]
  if (!msg.message) return

  const type = Object.keys(msg.message)[0]
  const isMedia = ['imageMessage', 'videoMessage',
                   'documentMessage', 'audioMessage'].includes(type)

  if (!isMedia) return

  const jid = msg.key.remoteJid
  await sock.sendMessage(jid, { text: '⏳ Mengupload...' })

  try {
    const buffer = await downloadMediaMessage(msg, 'buffer', {})
    const ext = type === 'imageMessage'   ? '.jpg'
              : type === 'videoMessage'   ? '.mp4'
              : type === 'audioMessage'   ? '.mp3' : '.bin'

    const url = await uploadToNyzz(buffer, `upload${ext}`)

    await sock.sendMessage(jid, {
      text: `✅ Upload berhasil!\n\n🔗 ${url}`
    })
  } catch (err) {
    await sock.sendMessage(jid, { text: `❌ Gagal: ${err.message}` })
  }
})
05

Integrasi Bot Telegram

📦 Contoh menggunakan node-telegram-bot-api. Install: npm install node-telegram-bot-api node-fetch
1
Setup bot & handler media
Bot akan mendengarkan kiriman foto, video, audio, dan dokumen, lalu otomatis upload dan balas dengan URL pendek.
javascript — node-telegram-bot-api
import TelegramBot from 'node-telegram-bot-api'
import fetch from 'node-fetch'
import FormData from 'form-data'

const bot = new TelegramBot('TOKEN_BOT_KAMU', { polling: true })
const UPLOAD_URL = 'https://uploader.nyzz.my.id/api/upload'

// Download file dari Telegram lalu upload ke NyzzUploader
async function uploadFromTelegram(fileId, filename) {
  const fileUrl = await bot.getFileLink(fileId)
  const dlRes   = await fetch(fileUrl)
  const buffer  = Buffer.from(await dlRes.arrayBuffer())

  const form = new FormData()
  form.append('file', buffer, { filename })

  const res  = await fetch(UPLOAD_URL, {
    method: 'POST',
    body: form,
    headers: form.getHeaders()
  })
  const json = await res.json()
  if (!json.success) throw new Error(json.error)
  return json.url
}

// Handler foto
bot.on('photo', async (msg) => {
  const chatId = msg.chat.id
  const photo  = msg.photo.at(-1) // resolusi tertinggi
  await bot.sendMessage(chatId, '⏳ Mengupload...')
  try {
    const url = await uploadFromTelegram(photo.file_id, 'photo.jpg')
    await bot.sendMessage(chatId, `✅ Berhasil!\n🔗 ${url}`)
  } catch (e) {
    await bot.sendMessage(chatId, `❌ Gagal: ${e.message}`)
  }
})

// Handler video
bot.on('video', async (msg) => {
  const chatId = msg.chat.id
  await bot.sendMessage(chatId, '⏳ Mengupload...')
  try {
    const url = await uploadFromTelegram(msg.video.file_id, 'video.mp4')
    await bot.sendMessage(chatId, `✅ Berhasil!\n🔗 ${url}`)
  } catch (e) {
    await bot.sendMessage(chatId, `❌ Gagal: ${e.message}`)
  }
})

// Handler audio & voice note
bot.on('audio', async (msg) => {
  const chatId = msg.chat.id
  await bot.sendMessage(chatId, '⏳ Mengupload...')
  try {
    const url = await uploadFromTelegram(msg.audio.file_id, 'audio.mp3')
    await bot.sendMessage(chatId, `✅ Berhasil!\n🔗 ${url}`)
  } catch (e) {
    await bot.sendMessage(chatId, `❌ Gagal: ${e.message}`)
  }
})

// Handler dokumen (semua format)
bot.on('document', async (msg) => {
  const chatId = msg.chat.id
  const doc    = msg.document
  await bot.sendMessage(chatId, '⏳ Mengupload...')
  try {
    const url = await uploadFromTelegram(doc.file_id, doc.file_name || 'file.bin')
    await bot.sendMessage(chatId, `✅ Berhasil!\n🔗 ${url}`)
  } catch (e) {
    await bot.sendMessage(chatId, `❌ Gagal: ${e.message}`)
  }
})
⚠️ Telegram membatasi download file gratis maks 20 MB. NyzzUploader sendiri membatasi 3 MB, jadi pastikan validasi ukuran sebelum upload.
